Herboria Shine - Netlify Email Notification Setup

1. Upload this ZIP to the SAME Netlify site.
2. In Netlify, go to Forms and enable form detection if it is not already enabled.
3. Open your website once and submit a test order.
4. In Netlify, go to Forms > herboria-order > Form notifications.
5. Add Email notification and set the recipient to herboriashine@gmail.com.
6. Submit another test order and check the inbox/spam folder.

No WhatsApp API token is required for this version.
